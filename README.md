# 📩 SMS Spam Classifier

Machine Learning web app that classifies SMS messages as Spam or Ham using Natural Language Processing.

## 🚀 Live Demo


## 📌 Features
- Detect Spam / Ham messages instantly
- NLP text preprocessing
- TF-IDF vectorization
- Trained ML model
- Clean web interface

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas
- NLTK
- Streamlit

## ⚙️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
